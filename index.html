Free Web Deployment Guide

Easily host this web application online so your entire team can access it on their smartphone or shared restaurant tablet.

1

Option A (Easiest) : Netlify Drop (Zero code drag & drop)

  1. Visit free website Netlify Drop (app.netlify.com/drop).
  2. Drag and drop this project folder containing index.html, css, and js.
  3. Your app goes live instantly with a secure HTTPS URL (e.g. https://resto-tips.netlify.app)!
2

Option B : GitHub + Vercel / GitHub Pages

  1. Create a free repository on GitHub.com.
  2. Push your files to the repository.
  3. Connect to Vercel.com or enable GitHub Pages in repo settings.
  4. Team members can add the web app link directly to their smartphone home screen!
3

Option C : Shared Restaurant Tablet Setup

  1. Open the live URL on your restaurant iPad or Android tablet.
  2. Tap "Add to Home Screen" in browser options to create a full-screen app shortcut.
  3. Staff can log tasks and view Coins and tip shares in seconds!
