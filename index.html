Guide de Déploiement Gratuit sur le Web

Pour que votre équipe et vous-même puissiez utiliser l'application sur smartphone ou tablette au restaurant sans installation.

1

Option A (La plus simple) : Déploiement sur Netlify (Drop sans code)

  1. Rendez-vous sur le site gratuit Netlify Drop (app.netlify.com/drop).
  2. Glissez-déposez le dossier de cette application (qui contient index.html, css et js).
  3. Votre application est en ligne en 5 secondes avec une URL sécurisée HTTPS (ex: https://resto-tips.netlify.app) !
2

Option B : Déploiement via GitHub + Vercel / GitHub Pages

  1. Créez un dépôt gratuit sur GitHub.com.
  2. Mettez les fichiers sur votre dépôt.
  3. Connectez votre dépôt à Vercel.com ou activez GitHub Pages dans les paramètres du dépôt.
  4. Toute l'équipe peut alors ajouter l'application sur l'écran d'accueil de son smartphone comme une application native !
3

Option C : Utilisation sur Tablette Commune du Restaurant

  1. Ouvrez l'URL générée sur la tablette du restaurant (ex: iPad / Android).
  2. Cliquez sur "Ajouter à l'écran d'accueil" dans le navigateur pour créer un raccourci d'application plein écran.
  3. Les employés peuvent y déclarer leurs tâches et consulter leurs présences et tips en un instant !
